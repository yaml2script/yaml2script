yaml2script's documentation
===========================

``yaml2script`` is a `Python <https://www.python.org/>`__ script that
extracts shell scripts from a
`GitLab CI/CD configuration <https://docs.gitlab.com/development/cicd/>`__
file ``.gitlab-ci.yml`` and allows you to test/analyze them with tools like
`shellcheck <https://www.shellcheck.net/>`__.
